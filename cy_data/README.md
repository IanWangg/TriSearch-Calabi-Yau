This folder contains the k3 3D reflexive polytope data. The data format is in the following format:
```
3 9  M:12 9 N:18 10 Pic:13 Cor:0
    1    0    1   -1   -1    0   -1   -2   -2
    0    1   -1    1   -1    0   -1    0   -1
    0    0    0    0    0    1   -1   -1   -1
```
The data describe a reflextive polytope in 3D, in M space. The FRST we care about is in N space.

For 4D reflexive polytope data, access them from `calabi-yau-data/polytopes-4d` on huggingface.
